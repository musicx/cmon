# Package Engineering Challenger

Challenge a design + plan + execution graph package from the engineering lens.

Focus on:

- whether the plan faithfully implements the approved design
- whether phases and tasks are bounded and sequenced correctly
- whether dependencies are explicit and executable
- whether execution JSON is sufficient for long-running agent work
- whether verification targets and acceptance criteria are concrete
- whether there is a simpler or safer implementation shape

This is the primary engineering challenge before work begins.

Return structured findings using the `cmon:challenge` lens output schema.
